````markdown
**版本：v1.1.1**

## 📘 程式簡介

**Diary GUI** 是一個使用 **Python + Tkinter** 製作的簡易圖形化日記應用程式。

它提供一個輕量、**離線** 的編輯環境，讓使用者能快速新增、編輯、刪除每日的日記內容。所有資料皆會**自動儲存在本地端的 JSON 檔案**中，無須網路即可使用。

此外，本程式內建 **GitHub 自動版本檢查機制**，可即時比對線上版本（`version.txt`），在有新版本時通知使用者前往下載。

## 💻 執行方式

### 📦 方法一：使用 Python 原始碼 (推薦開發者)

1.  確保您的環境已安裝 **Python 3.10 或以上版本**。
2.  安裝必要的函式庫（`requests` 用於檢查更新，`tkcalendar` 用於小日曆）：
    ```bash
    pip install requests tkcalendar
    ```
3.  執行主程式：
    ```bash
    python diary_gui.py
    ```

### 🪄 方法二：使用打包版 EXE (推薦一般使用者)

若您從 [GitHub Releases](https://github.com/yyy645815/diary/releases) 下載執行檔版本，只需**雙擊 `diary_gui.exe`** 即可使用。

## 🖥️ 操作介面詳細說明 (v1.1.1)

| 功能區域 | 按鈕/操作 | 功能說明 |
| :--- | :--- | :--- |
| **📅 日期清單 (左側)** | 點擊列表項目 | 載入該日內容到編輯區，**並自動儲存當前內容**。 |
| **📝 編輯區 (右側)** | 輸入文字 | 停止輸入約 1 秒後，觸發**自動儲存**至檔案。 |
| **新增日記** | `今天新日記` | 快速建立今日日記。 |
| **新增日記** | `新日記（指定日期）` | 透過對話框輸入日期。 |
| **日期選擇** | 點選日期輸入旁的 **日曆圖示** | 彈出日曆供視覺化選取日期。 |
| **資料管理** | `儲存本篇` / `儲存到檔案` | 立即強制手動儲存當前編輯內容。 |
| **資料管理** | `刪除這篇` | 移除該日期的日記資料。 |
| **列表維護** | `重新整理列表` | 手動更新並排序左側日記清單。 |
| **維護更新** | `檢查更新` | 連線檢查 GitHub 版本號，若有新版將跳出提示。 |

## 💾 資料儲存與風險提示

日記資料會自動儲存在同資料夾下的檔案：

> `diary_gui.json`

⚠️ **重要提醒：若手動刪除該檔案，舊日記資料將無法復原。**

### 🧩 檔案結構（Project Structure）

````

📁 diary/
├─ diary\_gui.py     \# 程式主邏輯
├─ version.txt      \# 線上版本號 (供程式比對更新使用)
├─ diary\_gui.json   \# 使用中自動產生，儲存所有日記資料 (請勿手動修改)
├─ .gitignore       \# 忽略不需上傳的檔案設定
└─ README.md        \# 專案說明文件 (您正在閱讀的文件)

