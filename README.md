# 📝 Diary GUI — 日記記錄程式

## 📘 程式簡介
Diary GUI 是一個使用 **Python + Tkinter** 製作的簡易圖形化日記應用程式。  
主要功能是提供一個輕量、離線的工具，讓使用者能快速新增、編輯、刪除每日的日記內容。  
所有資料皆會自動儲存在本地端的 JSON 檔中，無須連線即可使用。  

此外，本程式內建 **GitHub 自動版本檢查機制**，  
可即時比對線上版本（`version.txt`），在有新版本時通知使用者前往下載。

---

## ⚙️ 功能特色
- 📅 **日期管理**：以日期索引快速查詢與修改日記  
- 💬 **心情與內容記錄**：每日可記錄心情與文字內容  
- 💾 **自動儲存機制**：所有內容自動寫入 `diary_gui.json`  
- 🗑️ **刪除日記功能**：可移除指定日期的日記  
- 🔄 **版本檢查功能**：自動比對 GitHub 上的版本號  
- 🌙 **直覺 GUI 介面**：免指令操作，啟動即用

## 💻 執行方式

### 📦 使用 Python 原始碼
確保你的環境有安裝 Python 3.10 以上版本，並執行以下指令：

```bash
pip install requests

python diary_gui.py
```
🪄 使用打包版 EXE
若你從 Release 下載執行檔版本，只需 雙擊 diary_gui.exe 即可使用。

🖥️ 操作說明
功能區域	功能說明
📅 日期清單	顯示所有已建立的日記日期
📝 編輯區	可輸入或修改選取日期的內容與心情
💾 儲存按鈕	將日記內容寫入 JSON 檔
🗑️ 刪除按鈕	移除該日期的日記資料
🔄 檢查更新	連線檢查 GitHub 版本號，若有新版將跳出提示

📁 資料儲存
日記資料將自動儲存在同資料夾下的：
diary_gui.json
⚠️ 若手動刪除該檔案，舊日記資料將無法復原。

🔄 更新說明
