# 📝 Diary GUI — 日記記錄程式

## 📘 程式簡介

**Diary GUI** 是一個使用 **Python + Tkinter** 製作的簡易圖形化日記應用程式。

它提供一個輕量、**離線** 的編輯環境，讓使用者能快速新增、編輯、刪除每日的日記內容。所有資料皆會自動儲存在本地端的 **JSON** 檔案中，**無須網路即可使用**。

此外，本程式內建 **GitHub 自動版本檢查機制**，可即時比對線上版本（`version.txt`），在有新版本時通知使用者前往下載。

-----

## ⚙️ 核心功能一覽

| 類別 | 功能特色 | 說明 |
| :--- | :--- | :--- |
| **資料管理** | 📅 **日期管理** | 以日期索引快速查詢與修改日記。 |
| **內容記錄** | 💬 **心情與內容** | 每日可記錄心情與文字內容。 |
| **資料持久化** | 💾 **自動儲存機制** | 輸入後**自動**寫入 `diary_gui.json`，無需手動點擊。 |
| **介面操作** | 🗑️ **刪除日記功能** | 可移除指定日期的日記資料。 |
| **維護更新** | 🔄 **版本檢查功能** | 自動比對 GitHub 上的版本號。 |
| **使用者體驗** | 🌙 **直覺 GUI 介面** | 免指令操作，啟動即用。 |

## 💻 執行方式

### 📦 方法一：使用 Python 原始碼 (推薦開發者)

1.  確保您的環境已安裝 **Python 3.10 或以上版本**。
2.  安裝必要的函式庫：
    ```bash
    pip install requests
    ```
3.  執行主程式：
    ```bash
    python diary_gui.py
    ```

### 🪄 方法二：使用打包版 EXE (推薦一般使用者)

若您從 [GitHub Releases](https://www.google.com/search?q=GitHub%2520Releases) 下載執行檔版本，只需**雙擊 `diary_gui.exe`** 即可使用。

| 功能區域 | 功能說明 |

## 💾 資料儲存與風險提示

日記資料會自動儲存在同資料夾下的檔案：

> `diary_gui.json`

⚠️ **重要提醒：若手動刪除該檔案，舊日記資料將無法復原。**

### 🔄 版本更新機制

程式啟動時或點選「檢查更新」按鈕時，會連線讀取 GitHub 上的版本檔案 (`version.txt`) 以確認是否有新版。

  * **🔗 最新版本下載：** [GitHub Releases](https://www.google.com/search?q=GitHub%2520Releases)
